# Skills Pathfinder Documentation

## Overview

This directory contains the full design, architecture, and submission documentation for the **Skills Pathfinder** project, built for the FAU AI HootCamp Summer 2026.

## Documentation Index

- [**README.md**](../README.md): High-level project overview and quick start guide.
- [**ARCHITECTURE.md**](ARCHITECTURE.md): System architecture, data flow diagram, and component breakdown.
- [**DATABASE.md**](DATABASE.md): Supabase PostgreSQL schema, table definitions, and relationships.
- [**API.md**](API.md): API endpoint specifications for upload, skills, and matches.
- [**AI-DESIGN.md**](AI-DESIGN.md): Details on the hybrid AI strategy, including providers, prompting, and data validation.
- [**DESIGN.md**](DESIGN.md): Frontend design system, including colors, typography, and component guidelines.
- [**SECURITY.md**](SECURITY.md): Security measures covering authentication, authorization, and data privacy.
- [**TESTING.md**](TESTING.md): Manual test plan with key user scenarios and test cases.
- [**DEPLOYMENT.md**](DEPLOYMENT.md): Step-by-step guide for deploying the application to Vercel.
- [**GRADING-GATES.md**](GRADING-GATES.md): A checklist mapping project features to official grading requirements.

## How to Use

- Start with the main [project `README.md`](../README.md) for setup and a general introduction.
- Refer to `ARCHITECTURE.md` for a technical overview of how the system works.
